WASPrFlow Install Guide

Quick Steps

  1. Download installer from the WASPrFlow page.
  2. Run the setup EXE or MSI as Administrator.
  3. Complete prompts and launch WASPrFlow.

Troubleshooting

  1. If SmartScreen appears, choose More info, then Run anyway.
  2. If install fails, reboot and retry with Administrator rights.
  3. If needed, re-download installer and verify file size is non-zero.
