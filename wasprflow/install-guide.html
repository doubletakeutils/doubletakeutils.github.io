WASPrFlow Install And Setup Guide

This guide covers install, first launch, licensing, API keys, expected behavior, and troubleshooting for Windows users.

Recommended installer: Use Setup EXE for most users. Use MSI for managed business rollouts (RMM, Intune, SCCM, GPO workflows).

1. Download And Install

Step 1: Open the download page

Use either WASPrFlow MSI or WASPrFlow Setup EXE.

Step 2: If SmartScreen appears

Windows may show an unknown publisher prompt. Select Run anyway to continue installation.

Step 3: Complete setup wizard

Run through the setup wizard and finish installation.

2. First Launch And Main UI

Floating bar

The compact bar is your always-available control point. Use your configured hotkey to dictate and insert text.

Settings panel

Open settings from the bar. Configure hotkey, language, microphone, privacy mode, dictionary, and open licensing or API keys.

3. Licensing Workflow

Enter key or buy license

From the License panel, paste a key, import a saved .lic file, or purchase with your email.

License management after activation

After activation, this panel shows your licensed account and action buttons to remove, back up, or delete the local license file.

File loaded state

After import, the app confirms file load and enables activation with the loaded key.

Activated state

After successful activation, you should see Pro License and management actions for remove, backup, and delete.

Backup license file

Use Backup License to save a reusable file named WASPrflow_license.lic.

Expected Licensing Behaviors

Expected: Activation happens only after confirmed payment or valid key import/paste.
Expected: Saved license in AppData can be reused on startup through the license prompt flow.
Expected: You can import, backup, and delete local license files directly from the license panel.

4. API Keys

Built-in keys and custom keys

WASPrFlow includes built-in keys. Add your own Deepgram/OpenAI keys if you want dedicated quota, then click Save.

5. Expected Runtime Behavior

  • Hold configured hotkey to capture speech, release to insert text at current cursor position.
  • The bar reflects state changes (idle, listening, processing, success/error).
  • Very short utterances may be less reliable than full phrases depending on mic and noise conditions.
  • License purchase flow should send receipt/invoice and include license details based on configured payment worker setup.

6. Troubleshooting

  • Cannot activate: confirm purchase email matches, then try importing your saved .lic and activating again.
  • No receipt received: verify sender/domain settings in your payment email provider and check spam folder.
  • Mic issues: confirm selected input device in settings and Windows sound privacy permissions.
  • Punctuation quality: speak complete phrases and pauses; tune model settings as they are updated.
